# Fonts

This directory must contain **NotoNaskhArabic-Regular.ttf** for the Arabic annotator.

## Download

```bash
curl -L -o NotoNaskhArabic-Regular.ttf \
  "https://github.com/google/fonts/raw/main/ofl/notonaskharabic/NotoNaskhArabic%5Bwght%5D.ttf"
```

Or download manually from: https://fonts.google.com/noto/specimen/Noto+Naskh+Arabic

Why Noto Naskh: best free-software Arabic font for body text, SIL OFL license, clear at small sizes (important for bounding-box labels on cluttered X-rays).
